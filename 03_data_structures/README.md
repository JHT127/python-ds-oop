# Data Structures

Implementations of core data structures from scratch in Python, built progressively from linear to advanced non-linear structures.

| Folder | Structure | Key Operations |
|---|---|---|
| `linked_lists/singly_reverse/` | Singly Linked List | insert, delete, **reverse** |
| `linked_lists/sorted_circular/` | Circular Linked List | sorted insertion |
| `stacks/` | Array Stack | push, pop, peek |
| `queues/` | Linked Queue | enqueue, dequeue + simulation |
| `trees/binary_and_bst/` | Binary Tree + BST | insert, search, delete, traversals |
| `trees/avl_tree/` | AVL Tree | rotations, **range_search**, **kth_smallest** |
| `graphs/` | Adjacency List Graph | **BFS**, **DFS**, DFS post-order |
| `heaps/` | Max Heap | push, pop, **build_heap O(n)** |
| `hash_tables/` | Chaining Hash Table | insert, search, collision handling |

All structures are implemented without using Python's built-in collections (no `deque`, no `heapq`, etc.) to demonstrate understanding of the underlying mechanics.
